# Project Summary

## Title
Predictive Modeling for Course Demand and Revenue Forecasting on EduPro

## Objective
To build an analytics dashboard that helps EduPro forecast course demand, estimate revenue, and identify high-performing course categories.

## Deliverables
- Streamlit dashboard
- Clean GitHub repository files
- Executive overview
- Demand prediction module
- Revenue forecasting module
- Category analysis
- Feature impact analysis

## Notes
This version is compatible with Streamlit Editor and does not use scikit-learn.
